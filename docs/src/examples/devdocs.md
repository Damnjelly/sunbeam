# Devdocs (Bash / jq)

## Requirements

You will need to have [curl](https://curl.haxx.se/) and [jq](https://stedolan.github.io/jq/) installed.

## Usage

```bash
sunbeam ./devdocs.sh # List all docsets
sunbeam ./devdocs.sh <docset-slug> # List all entries for a docset
```
