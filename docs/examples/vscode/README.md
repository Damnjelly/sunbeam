# Search Recents VS Code Project

## Requirements

- [deno](https://deno.com)

## Installation

```sh
curl -L https://raw.githubusercontent.com/pomdtr/sunbeam/main/docs/examples/vscode/vscode.ts > ~/.local/bin/sunbeam-tldr
chmod +x ~/.local/bin/sunbeam-tldr
```

## Usage

```sh
sunbeam vsc
```
