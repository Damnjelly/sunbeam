# Extension Catalog

## [Mac Apps](https://github.com/pomdtr/sunbeam/tree/main/extensions/macapps.sh)

Open your favorite apps

### Platform

- `macos`

### Commands

- `list`: List All Apps

### Installation

```
sunbeam extension install https://raw.githubusercontent.com/pomdtr/sunbeam/main/extensions/macapps.sh
```
